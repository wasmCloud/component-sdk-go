# OpenTelemetry Exporter for WebAssembly Components

> [!IMPORTANT]  
> This requires TinyGo `0.35.0` (current `dev` branch) in order to compile.

`wasitel` provides a `wasi:http`-based OpenTelemetry Go exporter implementation.
